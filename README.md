# FizzBuzz-go

A simple implementation of fizzbuzz in go

To get and run:

```
git clone https://github.com/shahrzadnourzad/fizzbuzz-go.git
cd fizzbuzz-go
go build .
./FizzBuzz
```


To test:
```
go test
```

To see API doc (Swagger), after running the FizzBuzz, visit: http://localhost:8080/docs/index.html
